今日の「AIツール」検索Top結果から判明した勝利パターン:

**フックの型**:
- 「これは神。」（1文完結フック）
- 「〜するだけで〜できる」（簡単さを強調）
- 「〜は必見です」（ターゲット職業を明示）

**コンテンツ構造**:
1. 1文フック（インパクト重視）
2. ユースケースリスト（箇条書き3つ）
3. 「具体的に解説します👇」（続きへの引き込み）
4. 動画デモ（見てわかる実演が必須）

**バズのキーワード**:
- 「完全無料 + オープンソース」→ 最強ワード
- 「自分の家」「自分の地元」「通ってる大学」→ 個人への落とし込み
- 5.9Mツイートのルーツ: Minecraft×現実変換（ゲーマー×普通の人のクロスオーバー）

**注意点**:
- AI驚き屋リスク：両ツイートにコミュニティノートが付いた
- 「AIではない技術をAIと呼ぶ」は信頼失墜へ

**うちへの応用**:

### Twitter X アルゴリズム2026 深掘り調査

**新発見ポイント:**
- **Dwell Time (滞在時間)が最重要** → スレッド形式が最強。クリフハンガーで読み続けさせる設計
- **返信 > リポスト > いいね > ブックマーク** の順でアルゴリズム評価が高い
- **最初の30分が勝負** → 投稿後30分以内のエンゲージメントで拡散範囲が決まる
- **外部リンクはペナルティ** → リンクは投稿本文に入れず、リプライに入れる（「リンクはリプライへ」戦術）
- **X Premium（旧Twitter Blue）** → 4x in-network / 2x out-of-network の可視性ブースト
- **ニッチコミュニティ戦略** → "SIMクラスター"で同ジャンルアカウント群と繋がると関連ユーザーに優先表示
- **ハッシュタグは1-2個まで** → 多用するとスパム扱いでリーチ減
- **2-3時間以内の返信が重要** → コメントへの早期返信でアルゴ評価UP

**実装計画:**
- 投稿フォーマット変更: リンクはリプライに入れる
- スレッド形式を増やす（dwell time UP）
- フォロワー多めのAI系アカウントに投稿30分以内にリプライ
