# HP Builder Skill 再設計ノート

## 「顧客が喜ぶ」の分解
1. 初見で「おっ」と思わせる（3秒ルール）
2. 問い合わせ/予約が実際に増える（コンバージョン）
3. 「自分の店がこんなにカッコよく見える」という誇り
4. スマホで見ても崩れない安心感
5. Googleで見つかる（SEO）
6. 更新が簡単（運用コスト低い）
7. 競合と差別化できている

## 現状のスキルの弱点
- テンプレ感が強い
- 業種ごとの深い知見がない
- コンバージョン設計が甘い
- マイクロインタラクションが足りない
- パフォーマンス最適化がない
- SEO/OGP/構造化データがない
