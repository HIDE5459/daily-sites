## Twitter投稿フロー 完全確定（本日実証済み）

**成功した手順:**
1. `browser navigate` → `https://x.com/home`
2. `browser snapshot` → `textbox "Post text" [ref=eXX]` を取得
3. `browser act` kind:"click" ref=テキストエリア
4. `browser act` kind:"type" ref=テキストエリア text="..." （日本語・英語どちらもOK、slowly:trueは絶対NG）
5. `browser act` kind:"evaluate" fn:`() => { document.querySelector('[data-testid="tweetButtonInline"]').click(); return 'posted'; }`

**重要ルール:**
- 外部Playwrightスクリプトは使わない（CDP競合でハング）
- refは時間で切れる → 投稿ボタンはevaluateで直接クリック
- `slowly:true` → ブラウザクラッシュするので絶対禁止
- Twitter API Free tierは503多い → ブラウザ経由が安定
